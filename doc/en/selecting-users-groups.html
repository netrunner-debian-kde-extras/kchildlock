Add or delete users and groups

You select a new user or group by clicking on the icon with the small green plus sign. A window with the possibility to choose a user or a group from the list of available linux users or linux groups will appear. Pick the desired name.

After adding a new user or group, this user / group will be added to the list. Click on the desired user / group to edit the restrictions.

To delete the restricitions of user or group, highlight the desired name and click on the icon with big red cross. You have to confirm the deletion.
