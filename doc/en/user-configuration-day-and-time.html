The user configuration settings for day and time restrictions

On this page you configure the settings for up to five different users.

  • enable Restriction: enables the restrictions if checked. If not checked, all the other detailed restrictions on the lower part of the screen are ignored.

  • User Name: choose the user for whom the restrictions will be applied.

  • same restrictions for every day: if checked, the same restrictions for the specified user will be appplied to every weekday.

  • red and green time slider: this slider shows the allowed times for active sessions. The green area shows the allowed time range, In the red area there is no session allowd. You can move the whole range, or only the upper or lower limit by clicking with the mouse on the yellow border and drag it to the left or right. The time limits are displayed beneath the slider.

  • or max time: allowed time (measured in hours and minutes) for the user to have an active session per day.

  • restrictions depend on weekday: if checked, the restrictions for the specified user will be appplied to the individual days of the week.

  • Monday ... Sunday: if checked, the restrictions for the specified user will be appplied on this day of week. If not checked, there will be NO restriction on this day of week!

  • red and green time slider: allow active sessions in the green time range.

  • or max time: allowed time (measured in hours and minutes) for the user to have an active session on this weekday.

In the above example, user "martin" is allowed to have an active session on the computer from 10:00 am to 8:00 pm on mondays. The time limit is 2 hours 30 minutes. If he logs in at 11:00 am, he will be logged out at 1:30 pm. If he logs in at 7:00 pm, he will be logged out at 8:00 pm. The same restrictions are applied on tuesdays. On wednesdays he may log in earlier the morning. On Saturdays there are NO restrictions, he may use the computer as long as the maximal weekly usage is not reached... This weekly limit is 8 hours per week in this example. The values for the usage times are reset to zero the first time you log in on a new week.

If you want to disallow the usage on a certain weekday, limit the usage to a short time window fom e.g. 03:00 to 03:15
