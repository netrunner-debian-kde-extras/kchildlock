Chapter 4. User System Tray Information

If you enable the system tray icon for the user, he will get a appropriate icon in the system tray.

By doing a right mouseclick, he will see the following context menu:

After clicking on the menu "Show Restrictions" the current status will be displayed.

The displayed values are the current limitations, and the remaining time for today and for this week. The progress bar displays the percentage values for these times.
