Chapter 3. Configuration

The general configuration settings

On this page you configure the general settings.

  • The first checkbox is to enable or disable the restrictions globally.

  • The second checkbox is to enable or disable the display of the system tray icon. This icon informs the user about the used and remaining times.

  • The scan interval is the interval between two consecutive checks for the restrictions. By default the check will be executed once every minute.
