The settings for login time restrictions

HINT: The restrictions for a group will be applied to every member of the group. If the user has user restrictions too, both will be applied.

On the first tab you configure the login times for the highlighted user or group.

  • enable Restriction for this user: enables the restrictions if checked. If not checked, all the other detailed restrictions on the lower part of the screen are ignored.

  • Username: choose the user for whom the restrictions will be applied.

  • same restrictions for every day: if checked, the same restrictions for the specified user will be applied to every weekday.

  • red and green time slider: this slider shows the allowed times for active sessions. The green area shows the allowed time range, In the red area there is no session allowed. You can move the whole range, or only the upper or lower limit by clicking with the mouse on the yellow border and drag it to the left or right. The time limits are displayed beneath the slider.

  • restrictions depend on weekday: if checked, the restrictions for the specified user will be applied to the individual days of the week.

  • Monday ... Sunday: if checked, the restrictions for the specified user will be applied on this day of week. If not checked, there will be NO restriction on this day of week!

  • red and green time slider: allow active sessions in the green time range.

In the above example, user "martin" is allowed to have an active session on the computer from 10:00 am to 8:00 pm on Mondays. If he logs in at 7:00 pm, he will be logged out at 8:00 pm. The same restrictions are applied on Tuesdays. On Wednesdays he may log in earlier the morning. On Saturdays there are NO restrictions, he may use the computer as long as the maximal weekly usage is not reached...

If you want to disallow the usage on a certain weekday, limit the usage to a short time window from e.g. 03:00 to 03:15
